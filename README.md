局域网文件传输工具
简单易用的局域网内文件传输程序，同一网络下设备之间快速互传文件、文件夹，无需公网、无需第三方云服务。
功能特点：
纯局域网传输，速度快、无流量消耗
支持单文件 / 整个文件夹上传、下载
跨设备使用：电脑、手机同网均可访问
操作简单，开箱即用
